# Machine Learning Course Practice

This repository contains machine learning practice notebooks completed as part of a graduate-level Information Management course.

## Overview
The notebooks demonstrate hands-on experience with core machine learning concepts, including data preprocessing, model training, and evaluation using Python and Google Colab.

## Contents
- Supervised learning exercises (regression and classification)
- Basic model training and evaluation workflows
- Practical implementation using Jupyter/Colab notebooks

## Tools & Technologies
- Python
- Google Colab
- Scikit-learn
- Machine Learning
- Artificial Intelligence

## Purpose
This repository serves as technical evidence of AI and machine learning practice for academic study and AI-related competitions.
